# mqtt-benchmark

mqtt-benchmark -h host地址
               -p 端口号
               -u 登录用户名
               -P 登录密码
               -q QOS 消息质量
               -r retain 保留值
               -t 发布主题
               -m 消息内容
               -s 订阅主题
               -c 并发客户端数
               -tls ssl是否开启
               -k 连接超时时间
